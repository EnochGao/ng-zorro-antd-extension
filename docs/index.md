## 安装

首先要安装ng-zorro然后再安装扩展包

```bash
mpm install ng-zorro-antd-extension --save

```

|依赖|版本|
|---|----|
|angular|>=13.0.0|
|ng-zorro-antd|>=13.0.0|

该组件包是项目中所开发的组件，以用于生产环境，后续有扩展再添加，欢迎fork贡献代码
