表格表单。整个table为一个表单控件，适合当填写的表单为一个list集合的情况。

它输出的值为数组对象，形如：`[{},{}]`，组件内部实际每一行为一个`formGroup`，也可以用来做展示。

具体使用详情查看右侧示例。

引入

```ts
import { TableFormModule } from 'ng-zorro-antd-extension/table-form/table-form.module';

```
